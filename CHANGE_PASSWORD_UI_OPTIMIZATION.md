# 修改密码页面UI优化总结

## 🎨 优化内容

### 修改密码页面 (change-pwd.vue)

#### ✨ 新增功能
1. **标题图标**
   - 添加锁重置图标 `icon-ic-round-lock-reset`
   - 更醒目的视觉提示

2. **密码规则提示卡片**
   - 使用 NAlert 组件显示规则
   - 信息类型提示框
   - 清晰列出三条规则

3. **输入框图标**
   - 用户账号：`icon-ic-round-person`
   - 旧密码：`icon-ic-round-lock`
   - 新密码：`icon-ic-round-vpn-key`
   - 确认密码：`icon-ic-round-verified-user`

4. **密码强度指示器**
   - 实时计算密码强度
   - 三个等级：弱（红）、中（黄）、强（绿）
   - 显示在新密码输入框下方

5. **按钮图标**
   - 确认按钮：`icon-ic-round-check`
   - 返回按钮：`icon-ic-round-arrow-back`

### 密码登录页面 (pwd-login.vue)

   - 自定义样式类

## 📋 密码强度计算规则

```typescript
强度评分标准:
- 长度 >= 6位: +1分
- 长度 >= 10位: +1分
- 包含大小写字母: +1分 (或仅字母: +1分)
- 包含数字: +1分
- 包含特殊字符: +1分

强度等级:
- 弱 (1-2分): 红色标签
- 中 (3分): 黄色标签
- 强 (4-5分): 绿色标签
```
