## 🎯 密码强度算法

```typescript
评分规则:
1. 长度 >= 6位: +1分
2. 长度 >= 10位: +1分
3. 包含大小写字母: +1分（或仅有字母: +1分）
4. 包含数字: +1分
5. 包含特殊字符(@$!%*?&): +1分

强度等级:
- 弱 (≤2分): 红色
- 中 (3分): 黄色
- 强 (≥4分): 绿色
```
